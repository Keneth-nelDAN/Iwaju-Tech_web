# Les WebSockets
Les WebSockets protocole de communication réseau qui établit un canal de communication bidirectionnel et persistant, permettant aux clients et aux serveurs d'échanger des données en temps réel sans attendre une requête.
Contrairement au HTTP qui fonctionne sur un modèle requête-réponse où le serveur ne peut répondre qu'après une sollicitation du client, les WebSockets autorisent les deux parties à envoyer des messages à tout moment une fois la connexion établie.
Il est toujours initié par une requête HTTP puis la requête passe en mode WebSockets via le handshake pour établir la connexion bidirectionnel

# Technologies choisis

Backend : Node.js avec la bibliothèque Socket.io
